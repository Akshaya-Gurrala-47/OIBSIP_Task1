# Online Reservation System

## Internship Task
This project is developed as part of the Java Programming Internship at Oasis Infobyte.

## Project Description
The Online Reservation System is a console-based Java application that allows users to log in, reserve train tickets, and cancel reservations using a PNR number. The system stores reservation data during runtime and helps users manage bookings easily.

The main objective of this project is to demonstrate the use of Java programming concepts such as classes, objects, collections, and user input handling.

## Features

- User Login System
- Ticket Reservation
- Automatic PNR Number Generation
- Ticket Cancellation using PNR
- Reservation Details Display
- Menu-driven Console Interface

## Modules

### 1. Login Module
Users must enter a valid login ID and password to access the system.

Default Credentials:
- Login ID: admin
- Password: 1234

### 2. Reservation Module
Users can reserve a train ticket by entering the following details:

- Passenger Name
- Age
- Gender
- Train Number
- Train Name
- Class Type
- Date of Journey
- Source
- Destination

After successful reservation, a unique PNR number is generated.

### 3. Cancellation Module
Users can cancel their reservation by entering the PNR number. The system displays the reservation details and asks for confirmation before cancellation.

## Technologies Used

- Java
- OOP (Object Oriented Programming)
- Java Collections (HashMap)
- Scanner Class for user input
